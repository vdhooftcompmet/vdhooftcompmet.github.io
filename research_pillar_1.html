Machine learning to read metabolite structure from spectra
Research pillar I

In this pillar, we are working on computational metabolomics workflows for metabolome mining and annotation. Metabolome mining tools typically rely on large-scale mass spectral comparisons, and finding patterns in the metabolomics profiles. Such analyses can be used to structure metabolomics data in novel manners, for example by mass spectral networks in which structurally similar molecules cluster together based on their mass fragmentation spectra. Metabolite annotation tools typically aim to find exact matches in mass spectral libraries (library matching) or to locate structurally related molecules (i.e., analogues, analogue search) in the libraries. Our group is working on machine learning and deep learning supported workflows to improve the performance both in accurary and in scalability of the metabolite annotation step. Here, we draw inspiration from natural language processing where various text-mining algorithms for topic modeling and word embedding serve as a springboard to find relevant patterns in metabolomics data. Once an increasing number of those patterns will be annotated by the community, large-scale comparisons using these patterns become even more powerful.

Our contributions to this challenging and exciting field

Advances in decomposing complex metabolite mixtures using substructure- and network-based computational metabolomics approaches 
This computational metabolomics review provides an in-depth overview of various substructure-based and network-based approaches that improve metabolite annotation rate and facilitate the biochemical interpretation of mass spectrometry and nuclear magnetic resonance spectroscopy metabolomics profiles.

Topic modeling for untargeted substructure exploration in metabolomics 
This foundational paper applies natural language processing algorithms for the first time to metabolomics data to discover substructure patterns imprinted in mass fragmentation spectra.
